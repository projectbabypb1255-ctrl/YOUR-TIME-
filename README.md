Crypto Investment Store
Welcome to the Crypto Investment Store – a modern, sleek, and interactive website designed for crypto enthusiasts and investors!
🌟 Features
🔐 Secure Gmail Login: Users can log in safely using their Google account via Firebase Authentication.
📊 Real User Dashboard: Track total investments, earnings, and referral bonuses in real-time.
💸 MoMo Payment Ready: Seamless MTN Mobile Money integration for product purchases.
🪙 10 Investment Products: Each product shows price and expected income, with automatic earnings calculation.
🔗 Referral System: Share your unique referral link and earn bonuses for every new user you invite.
🌐 Animated Bitcoin Background: Eye-catching moving Bitcoin icons to give a professional crypto vibe.
📱 Responsive Design: Works perfectly on desktop, tablet, and mobile devices.
🚀 Purpose
This website provides a fun, interactive, and educational experience for anyone looking to explore crypto investment simulations or build a crypto product platform.
It’s built to showcase real-time features like login, purchases, dashboard tracking, and referrals, providing a professional user experience.
🛠 Technologies Used
HTML, CSS, JavaScript – Core front-end development
Firebase – Authentication and Firestore database
MTN MoMo API (Demo) – Payment integration placeholder
Responsive Design – Mobile-first approach
🎯 Goal
The goal of this project is to provide a fast, sleek, and positive user experience while demonstrating the potential of crypto product platforms. Users can track investments, explore referrals, and experience a professional dashboard, all in a visually appealing interface.
